### 튜플의 모든 값을 가져오기

타입스크립트도 인덱스를 사용해서 배열의 값에 접근할 수 있다. 그런데 특정한 숫자 대신 `number`를 주면 모든 인덱스의 값을 유니언으로 나열하게 된다.
```ts type.ts mark=6[49:56]
interface UserRoleConfig {
  user: ["view", "create", "update"],
  admin: ["view", "create", "update", "delete"]
}

type Role = UserRoleConfig[keyof UserRoleConfig][number]
// type Role = "view" | "create" | "update" | "delete"
```

이 경우 `UserRoleConfig`의 모든 키에 접근하고, 배열인 모든 값을 나열하게 된다.
