### 유니언 타입에서 특정값을 없애거나 대체하기

제목 그대로다. 유니언 타입에서 특정 값을 없애거나 대체할 필요가 있(을 수 있)다.  
이때 쓰이는 것이 제네릭과 `extends`다.
<br/>
- 없애는 상황  
"제네릭으로 받는 타입에 `"c"`가 있으면 걔는 없는(`never`) 걸로 해주세요. 나머지는 그대로 주시고요" 
```ts type.ts
export type Letters = "a" | "b" | "c";

type RemoveC<T> = T extends "c" ? never : T;
type Result = RemoveC<Letters>
// Result = "a" | "b"
```

<br/>
- 대체하는 상황  
"제네릭으로 받는 타입에 `"c"`가 있으면 걔는 `메롱`으로 주세요. 나머지는 그대로 주시고요" 
```ts type.ts
export type Letters = "a" | "b" | "c";

type ReplaceC<T> = T extends "c" ? "메롱" : T;
type Result = ReplaceC<Letters>
// Result = "a" | "b" | "메롱"
```
